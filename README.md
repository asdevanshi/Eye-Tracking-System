# Eye Tracking System

## Overview

This project is a Python-based eye tracking system that detects pupil movement and estimates gaze direction using computer vision.

## Features

- Eye detection
- Pupil tracking
- Gaze estimation
- Alarm system
- Gaze data logging

## Technologies Used

- Python
- OpenCV
- NumPy

## Project Structure

- main.py
- 2dgaze.py
- Eye pupil gaze.py
- grazealarm.py
- the logger.py
- the range.py

## Future Improvements

- Improved calibration
- Real-time visualization
- Machine learning based gaze estimation
